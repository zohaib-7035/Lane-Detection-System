# Lane Detection System

## Overview

This repository contains a lane detection system implementation. The system is designed to identify lanes on the road using computer vision techniques.

## Features
- Real-time lane detection
- Robust against different lighting conditions
- High accuracy

## Requirements
- Python 3.x
- OpenCV
- NumPy

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/zohaib-7035/Lane-Detection-System.git
   ```

2. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
You can run the lane detection script with the following command:
```bash
python lane_detector.py
```
